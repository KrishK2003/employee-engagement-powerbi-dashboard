# Employee Engagement Dashboard

## Project Overview
This Power BI dashboard analyzes employee engagement, satisfaction, and work-life balance to help organizations make data-driven HR decisions.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Key Features
- Employee Satisfaction KPIs
- Work-Life Balance Analysis
- Interactive Filters and Slicers
- Workforce Trend Analysis
- HR Performance Metrics

## Key Insights
- Identified factors affecting employee satisfaction.
- Analyzed work-life balance trends across departments.
- Developed interactive visualizations for business stakeholders.

## Skills Demonstrated
- Data Cleaning
- Data Visualization
- Dashboard Development
- Business Intelligence
- KPI Reporting
